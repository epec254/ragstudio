Databricks Runtime 11.x migration guide

This guide helps you migrate your Databricks workloads to the latest version of Databricks Runtime 11.x. To do so, Databricks recommends that you migrate your workloads in the following order:

  • If your workloads are currently running on Databricks Runtime 10.4 LTS or above, you can migrate directly to the latest version of Databricks Runtime 11.x, as described later in this article.

  • If your workloads are currently running on Databricks Runtime 10.3 or below, do the following:

    1. Migrate to Databricks Runtime 10.4 LTS first. See the Databricks Runtime 10.x migration guide.

    2. Follow the guidance in this article to migrate from Databricks Runtime 10.4 LTS to the latest version of Databricks Runtime 11.x.

Note

LTS means this version is under long-term support. See Databricks Runtime LTS version lifecycle.

For information on migrating between Databricks Runtime versions, see the Databricks Runtime migration guide.
