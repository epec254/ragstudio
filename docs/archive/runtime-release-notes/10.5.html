Databricks Runtime 10.5 (unsupported)

The following release notes provide information about Databricks Runtime 10.5, powered by Apache Spark 3.2.1. Databricks released these images in May 2022.

New features and improvements

  • SQL: New functions to_number and try_to_number

  • SQL: New function try_sum

  • SQL: Update to ceil, ceiling, and floor functions

  • SQL: New aggregate functions regr_avgx and regr_avgy

  • SQL: Update to contains, startswith, and endswith functions

  • SQL: New function array_size

  • Unity Catalog: SQL LIST output removes is_directory column and adds trailing / for directory paths (Public Preview)

  • Auto Loader: new SQL function CLOUD_FILES_STATE

  • Auto Loader: file notification support is now GA

  • Delta Lake: new maxRecordsPerFile option for maximum records written to a single file

  • Deprecation of Koalas

  • Google BigQuery connector: Arrow compression codec for reads is now configurable

  • New _metadata column for all file formats

  • Test your Delta Live Tables pipeline with Databricks Runtime 10.5

SQL: New functions to_number and try_to_number

The to_number function and try_to_number function allow you to convert character representations of numbers in various formats to DECIMAL.

SQL: New function try_sum

Unlike sum, the try_sum aggregate function returns NULL instead of an error when an overflow occurs.

SQL: Update to ceil, ceiling, and floor functions

The ceil function, ceiling function, and floor function now support a second optional argument to specify a target scale.

SQL: New aggregate functions regr_avgx and regr_avgy

The new regr_avgx aggregate function and regr_avgy aggregate function support linear regressions.

SQL: Update to contains, startswith, and endswith functions

The contains function, startswith function, and endswith function now support binary data types as input in addition to strings.

SQL: New function array_size

The new array_size function calculates the size of an array.

Unity Catalog: SQL LIST output removes is_directory column and adds trailing / for directory paths (Public Preview)

The output of the SQL LIST statement for listing paths governed by Unity Catalog has changed. The is_directory column is no longer displayed. Instead, paths that represent directories now end with a trailing slash (/) for the path and name columns. This change is for consistency with the output of the dbutils.fs.ls command.

Auto Loader: new SQL function CLOUD_FILES_STATE

You can use the new CLOUD_FILES_STATE function to query the internal state of an Auto Loader stream; see Querying files discovered by Auto Loader.

Auto Loader: file notification support is now GA

File notification support for What is Auto Loader? on Google Cloud is now generally available.

Delta Lake: new maxRecordsPerFile option for maximum records written to a single file

When you use the DataFrame APIs to write to a Delta table, you can use the maxRecordsPerFile option to specify the maximum number of records to write to a single file. Specifying a value of zero or a negative value represents no limit. See Configure Delta Lake to control data file size.

Deprecation of Koalas

Koalas is now deprecated. Koalas provides pandas APIs on top of Apache Spark. Koalas was contributed to by the community and has been available as pandas API on Spark beginning in Apache Spark 3.2. Koalas continues to be preinstalled for Databricks Runtime 10.5, but Koalas will no longer be preinstalled beginning with the next Databricks Runtime version.

To learn more about pandas API on Spark, see:

  • Pandas API on Spark in the Databricks documentation

  • Pandas API on Spark in the Apache Spark documentation

  • Migrating from Koalas to pandas API on Spark in the Apache Spark documentation

Google BigQuery connector: Arrow compression codec for reads is now configurable

You can now configure the Arrow compression codec that the Google BigQuery connector uses for reads, by setting the arrowCompressionCodec DataFrame option. Available options include:

  • COMPRESSION_UNSPECIFIED: no compression is used (the default)

  • ZSTD: Zstandard compression

  • LZ4_FRAME: See LZ4 Frame Format Description

New _metadata column for all file formats

You can get metadata information for input files with the _metadata column. The _metadata column is hidden. This column is available for all input file formats. To include the _metadata column in a returned DataFrame, you must explicitly reference this column in your query. See File metadata column.

Test your Delta Live Tables pipeline with Databricks Runtime 10.5

Databricks Runtime 10.5 is now the supported preview version for Delta Live Tables. To test your pipeline with Databricks Runtime 10.5, set the channel setting to preview in your pipeline settings.

Bug fixes

  • Fixed the display of numUpdatedRows and numCopiedRows metric results for update operations on Delta tables that have change data capture (CDC) enabled.
